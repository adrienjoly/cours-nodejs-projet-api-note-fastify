# esgi-nodejs-projet
Cours Node.jsCours Node.js - Réalisation d'une API pour un client


## Installation

- Clonner le repos
```
    git clone https://github.com/dd701116/esgi-nodejs-projet.git
```

- Installer les dépendances
```
    cd esgi-nodejs-projet
    npm install
```

- Lancer le serveur
```
    npm start
```
